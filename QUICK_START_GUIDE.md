## Troubleshooting ğŸ”§

### "Can't request refund"
- Only **Delivered** or **Accepted** orders can be refunded
- Already refunded orders can't be refunded again

### "Can't cancel order"
- Only **Pending** orders can be cancelled
- Use refund request for accepted/delivered orders

### "Track Order not showing history"
- Order may be newly created
- History builds as status changes

### "Refund button not appearing"
- Check order status (must be Delivered or Accepted)
- Refresh the page if needed
